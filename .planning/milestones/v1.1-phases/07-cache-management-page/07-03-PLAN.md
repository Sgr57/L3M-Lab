Named export `ModelDownloader` with prop: `onDownloadComplete?: () => void`

**State:**
```typescript
const [query, setQuery] = useState('')
const [results, setResults] = useState<HFModelResult[]>([])
const [isOpen, setIsOpen] = useState(false)
const [loading, setLoading] = useState(false)
const [selectedModel, setSelectedModel] = useState<HFModelResult | null>(null)
```

Also read from compare store for download state:
```typescript
const executionStatus = useCompareStore((s) => s.executionStatus)
const downloadProgress = useCompareStore((s) => s.downloadProgress)
const isDownloading = executionStatus === 'downloading'
```

**Search logic (reuse from ModelSelector pattern):**
- useEffect on `debouncedQuery`: call `searchModels(debouncedQuery)`, set results, set isOpen
- useEffect for outside click: close dropdown on mousedown outside wrapperRef
- Results dropdown shows identical UI to ModelSelector dropdown (model ID bold, pipeline tag, ONNX badge, download count, likes count)

**Model selection:**
- `handleSelectModel(model)`: set selectedModel, close dropdown, clear query, fetch `fetchModelDetails(model.modelId)`, set modelDetails, set selectedQuant to default (prefer 'q4' if available, else first)

**Post-selection UI (per UI-SPEC):**
- Card showing selected model name
- Quantization pills row: each quantization from modelDetails as a button
  - Unselected: `rounded-lg border border-border px-2 py-1 text-xs`
  - Selected: `rounded-lg border border-primary bg-webgpu-bg px-2 py-1 text-xs font-semibold text-primary`
  - Show estimated size next to each pill if available from `modelDetails.sizeByQuant`
- "Download Model" button: `rounded-lg bg-primary px-4 py-2 text-xs font-semibold text-white disabled:opacity-40 disabled:cursor-not-allowed`
- Button disabled during download or if no quantization selected
- Button text: "Downloading..." when isDownloading, "Download Model" otherwise

**Download handler:**
- `handleDownload()`:
  1. Create a temporary TestConfig for the download:
     ```typescript
     const config: TestConfig = {
       id: `download-${selectedModel.modelId}-${selectedQuant}-${crypto.randomUUID()}`,
       modelId: selectedModel.modelId,
       displayName: selectedModel.name,
       quantization: selectedQuant,
       backend: webgpuSupported ? 'webgpu' : 'wasm',
       estimatedSize: modelDetails?.sizeByQuant[selectedQuant] ?? 0,
       cached: false,
     }
     ```
  2. Call `startDownload([config])`
  3. The download progress will be shown via the existing download progress mechanism in useCompareStore

- Read `webgpuSupported` from useSettingsStore: `const webgpuSupported = useSettingsStore((s) => s.webgpuSupported)`

**Download progress display:**
- When isDownloading and downloadProgress exists, show progress bar below the Download button:
  ```tsx
  <div className="mt-3">
    {downloadProgress.models.map((model) => (
      <div key={model.configId} className="flex items-center gap-3 text-xs">
        <span className="w-48 truncate text-text-primary">{model.modelName}</span>
        <div className="flex-1">
          <div className="h-1.5 w-full rounded-full bg-border-light overflow-hidden">
            <div
              className={`h-full ${model.status === 'complete' ? 'bg-success' : 'bg-primary'}`}
              style={{ width: `${model.status === 'complete' ? 100 : Math.min(model.progress, 100)}%` }}
            />
          </div>
        </div>
        <span className="w-12 text-right text-text-secondary text-[11px]">
          {model.status === 'downloading' && `${Math.round(model.progress)}%`}
          {model.status === 'complete' && <span className="text-success">Done</span>}
        </span>
      </div>
    ))}
  </div>
  ```

**Post-download: trigger refresh**
- Use a useEffect that watches `executionStatus`: when it transitions from 'downloading' to 'idle' (download complete), call `onDownloadComplete?.()` and reset selectedModel/modelDetails/selectedQuant to null
  ```typescript
  const prevStatusRef = useRef(executionStatus)
  useEffect(() => {
    if (prevStatusRef.current === 'downloading' && executionStatus === 'idle') {
      onDownloadComplete?.()
      setSelectedModel(null)
      setModelDetails(null)
      setSelectedQuant(null)
    }
    prevStatusRef.current = executionStatus
  }, [executionStatus, onDownloadComplete])
  ```
