Create `src/components/CachedModelsTable/index.tsx` as the primary cache management component.

**Component structure:**
- Named export `CachedModelsTable`
- Takes an optional prop `onCacheChanged?: () => void` (callback to notify parent when cache is modified, so ModelDownloader can trigger re-enumeration)

**State management:**
```typescript
const [models, setModels] = useState<CachedModelInfo[]>([])
const [loading, setLoading] = useState(true)
const [error, setError] = useState<string | null>(null)
const [expanded, setExpanded] = useState<Set<string>>(new Set())  // set of modelIds
const [sortKey, setSortKey] = useState<'modelId' | 'totalSize' | 'lastUsed'>('modelId')
const [sortDir, setSortDir] = useState<'asc' | 'desc'>('asc')
const [deleting, setDeleting] = useState<string | null>(null)  // "modelId" or "modelId::quant" being deleted
```

**Cache enumeration on mount:**
- In a `useEffect([], ...)`, call `loadCache()` async function
- `loadCache`: set loading=true, call `enumerateCache()`, then `groupByModelAndQuant(entries, useModelUsageStore.getState())`, set models, set loading=false
- Wrap in try/catch, set error message on failure: "Unable to read browser cache. Make sure you're using HTTPS and a supported browser."
- Export the `loadCache` function via a ref or expose it via `useImperativeHandle` — OR simpler: use a `refreshKey` counter state that increments to trigger re-enumeration via useEffect dependency

Actually, simplest approach: create a `loadCache` function inside the component and call it from useEffect. Also expose a `refresh` mechanism: use a `refreshCounter` state, increment it after deletions and in the `onCacheChanged` callback. The useEffect depends on `[refreshCounter]`.

**Sorting logic:**
- `handleSort(key)`: if same key, toggle direction; if different key, set new key with default direction (asc for modelId, desc for totalSize and lastUsed)
- Apply sort to models before rendering: 
  - `modelId`: localeCompare
  - `totalSize`: numeric comparison
  - `lastUsed`: treat null as -Infinity (sorts to end for desc, beginning for asc)
- Sort indicator: append ` ↑` or ` ↓` to the active column header text

**Delete handlers:**
- `handleDeleteQuant(modelId, quantization)`: 
  1. `window.confirm('Delete ${quantization} cache for ${modelId.split('/').pop()}? This cannot be undone.')`
  2. If confirmed: set deleting to `${modelId}::${quantization}`, call `deleteCachedModel(modelId, quantization)`, call `useModelUsageStore.getState().removeUsage(modelId, quantization)`, increment refreshCounter, set deleting to null
- `handleDeleteModel(modelId)`:
  1. `window.confirm('Delete all cached files for ${modelId.split('/').pop()}? This cannot be undone.')`
  2. If confirmed: set deleting to modelId, for each quantization in the model call `deleteCachedModel(modelId, q.quantization)`, then call `deleteCachedModel(modelId)` once more to clean up shared files, call `useModelUsageStore.getState().removeUsage(modelId)`, increment refreshCounter, set deleting to null
  3. Call `onCacheChanged?.()` after deletion

**Cleanup handler:**
- `handleCleanup()`:
  1. Get stale keys via `getStaleModelKeys(models, 14 * 24 * 60 * 60 * 1000)`
  2. Calculate total size of stale entries from models data
  3. `window.confirm('Delete ${staleKeys.length} model(s) not used in over 2 weeks? This frees approximately ${formatSize(totalStaleSize)}.')`
  4. If confirmed: for each stale key, call `deleteCachedModel(key.modelId, key.quantization)` and `useModelUsageStore.getState().removeUsage(key.modelId, key.quantization)`, increment refreshCounter
  5. Call `onCacheChanged?.()` after cleanup

**Relative time formatting:**
- Create a helper function `formatRelativeTime(timestamp: number): string` inside the component file:
  - Calculate `diffMs = Date.now() - timestamp`
  - Less than 60s: "just now"
  - Less than 60min: `${Math.floor(diffMs / 60000)} minutes ago`
  - Less than 24h: `${Math.floor(diffMs / 3600000)} hours ago`
  - Less than 7d: `${Math.floor(diffMs / 86400000)} days ago`
  - Less than 30d: `${Math.floor(diffMs / 604800000)} weeks ago`
  - Otherwise: `${Math.floor(diffMs / 2592000000)} months ago`
